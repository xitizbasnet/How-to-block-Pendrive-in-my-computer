# How-to-block-Pendrive-in-my-computer
How to block Pendrive in my computer

-------------

1. Press Win + R

-----------

2. Type = "regedit" and hit enter

-----------

3. Go to the:
 
   Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\USBSTOR 
   
   and go to the "Start"
   
--------
   
5. Change the value from "3" to "4" 

----------
